My first non-blog-related project for my apprenticeship was to create a semantic HTML version of a restaurant menu. CSS could be used for simple styles and positioning, but no images could be used. I would use HTML elements to make the code simple, and easy to read and comprehend.

While this project seems simple, it took a lot more thought than I figured it would require.

Teaching myself HTML and CSS was a bit of a trainwreck composed of hacking bits of code together and having it work--but just barely. So I knew that learning to write everything myself from top to bottom would be a real challenge.

For this project, I read Exressive HTML by Billy Whited. I basically started by just writing up a simple HTML document, as I normally would. It was funny to go back and see how the way that I write my HTML mimics the way I would hack different pieces of HTML code together to make sites---disjoined and ungraceful. With the help of my mentor, I was able to make my code more concise and flow a lot better.
